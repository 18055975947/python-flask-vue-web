## python-flask-vue-web

> 使用Flask和vue-cli脚手架，开发vue3.x版本的python web 项目

- vue-cli：@vue/cli 4.5.7
- python：Python 3.7.0
- flask：1.1.2

### 项目构建

```python
# 在 pyCharm 创建项目，自带 python 虚拟环境

# 开发前端项目
cd vue-web
yarn

# 打包前端项目
cd vue-web
yarn run build

# python
pip install flask
pip install flask_cors

```
